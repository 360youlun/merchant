# Django-Merchant

Django-Merchant is a django application that enables you to use
multiple payment processors from a single API.

## Gateways
Following gateways are supported:

### Credit Card Processing

 * Authorize.net
 * Paypal
 * Eway
 * Braintree Payments

### Off-Site Processing

 * Paypal
 * RBS WorldPay
 * Google Checkout
 * Amazon FPS
